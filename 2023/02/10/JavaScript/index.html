发表于 2023-02-10 更新于 2023-03-16 分类于 JavaWeb

JavaScript

内部脚本

Script可以放置于任意地方，数量无限制，一般情况都会放到BODY标签底部，改善显示速度

外部脚本

SRC为JS文件URL路径，

书写语法

每行结尾分号可有可无

输出语句：

alert：写入警告框

write：写入HTML

变量

弱类型的值，可以存放任意数据的值，仅需要使用var声明一下

var text = 20；

text= “张三”

Var定义的变量相当于全局变量

变量可以重复定义

Let定义的变量相当于局部变量，且变量不可以重复定义

console定义的变量相当于常量，定义后不能被改变

数据类型

5种原始类型，1种引用类型

原始类型：

number：注意NaN与任何数相加结果仍为NaN

string

boolean

null

undefined

在JS中，进行比较时，会先进行类型转换，再去比较对应的值

”===“全等于不会进行类型转换

类型转换

字符串转换为数字，若字面不是数值，则转换成NAN

BOOLEAN类型转数字，TRUE转为1，False转为0

其他类型转BOOLEAN类型

NUMBER：0或NaN转换为false，其余数字转为true

String：空字符串转换为false，其余的转换为true

Null：转换为false

Undefined：转换为false

方法定义

Function 方法名（方法参数）{

}

JS中不需要定义返回参数类型，形参也不需要定义类型

对象

基本对象

Array：

Var 变量名 = new Array（元素列表）；

Var 变量名 = [ 元素列表 ] 注意为中括号

数组访问与JAVA一样

特殊点：JS中的数组是变长变类型的，即既可以存整数，也可以存字符串

增加元素 .Push

删除元素 .splice（Strat number，删除元素数量）

字符串

trim（）去除字符串两端空白字符

自定义对象

var 对象名称 = {

属性名称：属性值，

属性名称：属性值，

函数名称：function（）{

}

}

BOM对象

Window对象

带有确认和取消的对话框，带有返回值，返回true或false

定时器

setTimeOut不会循环执行

History对象

访问Window.history.方法();

BACK 后退

FORWARD 前进

Location对象

Window.location；

href获取或设置当前地址栏的URL

location.href = …..

Dom对象

获取Element对象

1.getElementById 返回一个Element对象

2.getElementByTagName 返回一个Element类型数组

3.getElementByName 返回一个Element类型数组

4.getElementByClassName 返回一个Element类型数组

事件监听

事件绑定

onblur 元素失去焦点

onfocus 元素获得焦点

onclick 点击事件

onkeydown 某个按键被按下

onmouserover鼠标移到某个元素

onmouseout鼠标移出某个元素

onsubmit submit按键被按下
